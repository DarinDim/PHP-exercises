# Администраторски панел

Това е администраторски панел с логин система и управление на различни видове данни.

## Характеристики

- **Логин система** - защита на администраторския раздел
- **Управление на продукти** - добавяне, редактиране, изтриване
- **Управление на категории** - организиране на продукти по категории
- **Заявки от потребители** - преглед на поръчки
- **Сесии** - безопасна работа с потребител

## Начало

1. Конфигурирайте базата данни в `db.php`
2. Посетете `http://localhost/exercises/12_admin_panel/`
3. Логнете се с администраторски акаунт
4. Используйте менюто за навигация между различните секции

## Структура

```
12_admin_panel/
├── index.php              # Начало (dashboard)
├── login.php              # Логин страница
├── logout.php             # Изход
├── db.php                 # Връзка с база
├── products.php           # Списък продукти
├── add-product.php        # Добавяне продукт
├── edit-product.php       # Редактиране продукт
├── delete-product.php     # Изтриване продукт
├── categories.php         # Управление категории
├── orders.php             # Поръчки
└── style.css              # CSS стилове
```

## Функции (опционално)

- Редактиране на продукти
- Пагинация (10 продукта на страница)
- Управление на администратори
- Отчети и статистика
