# Упражнение 15 — Форма с Regex валидация

Цел: да се създаде добре изглеждаща форма с поне две различни полета и да се приложи валидация с регулярни изрази (Regex).

Описание:
- Поля: **Име** (само букви, 3–50 символа) и **Имейл**.
- Сървърна проверка с `preg_match()`.
- Грешките се показват под съответното поле.
- Лека CSS стилизация за приятен интерфейс.

Примерни regex изрази използвани в кода:
```
/^[A-Za-z\- ]{3,50}$/u   // име
/^[\w\.-]+@[\w\.-]+\.[A-Za-z]{2,6}$/   // имейл
```
