# Каталог на продукти

Това е пълен каталог на продукти със следните функции:

## Съдържание

- **Цел**: Демонстриране на e-commerce система с продукти
- **Използвани технологии**: PHP, MySQL, HTML, CSS
- **Функции**:
  - Преглед на продукти
  - Кошница за пазаруване
  - Регистрация и логин на потребители
  - Администраторски панел за управление на продукти
  - Заявки за поръчки
  - PDO за безопасност

## Инсталация

1. Импортирайте `seed.sql` в MySQL
2. Конфигурирайте `db.php` с вашите данни за база
3. Посетете `http://localhost/exercises/11_product_catalog/`

## Структура

```
11_product_catalog/
├── index.php           # Начална страница
├── db.php             # Конфигурация на база данни
├── admin/             # Администраторски раздел
├── customer/          # Потребителски раздел
└── seed.sql           # SQL за таблици
```

## Отбележи

За пълната версия, вижте папката `/catalog` в корена.
