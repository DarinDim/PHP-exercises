# Упражнение 17 — Калкулатор с PHP

Цел: да се създаде калкулатор с уникален дизайн, който записва всеки въведен израз във файл.

## Какво прави упражнението
- Изработва се калкулатор с HTML/CSS/JS интерфейс
- Изразът се изпраща към PHP за обработка
- Резултатът се изчислява на сървъра
- Във файла `calculator_history.txt` се записва всеки въведен израз и резултат
- Показват се последните запазени операции

## Подробности
- Позволени символи: цифри, десетични точки, +, -, *, /, %, скоби
- Невалидните изрази се блокират
- При успешен изчислителен резултат се добавя нов ред във файла с дата и час

## Файлова структура
```
17_Calculator/
├── index.php              # Основният калкулатор с логиката
├── README.md              # Това описание
└── calculator_history.txt # Файл със записаните операции (генерира се автоматично)
```
