
# Упражнение 14 — Супергерои (Bootstrap)

Тема: Супергеройска промо страница. Страницата представя различни супергерои чрез карти и слайдшоу.

Използвани Bootstrap компоненти:
- **Navbar** — горна навигация с мобилен бутон
- **Carousel** — слайдшоу за представяне на главни персонажи
- **Card** — отделни карти за всеки супергерой
- **Grid (row / col)** — адаптивно подреждане на картите

Файлова структура:
- `index.php` — главната страница 
- `images/` — папка за вашите лични снимки на супергерои
