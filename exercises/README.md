# 📚 PHP Упражнения - Колекция

Това е пълна колекция от **15 PHP упражнения**, организирани по сложност и раздели.

## 📋 Структура на упражненията

### 🔢 Упражнения 1-2: Цикли
- **01_numbers_even_odd** - Отпечатване на числа с условна форматизация
- **02_square_with_loops** - Рисуване на квадрат със вложени цикли

### 📊 Упражнения 3-5: Масиви и Сесии
- **03_arrays_random_numbers** - Масив със случайни числа и цветна разлика
- **04_arrays_colors_divs** - Div тагове със цветни фонове и адаптивен текст
- **05_arrays_session** - Работа със PHP сесии за съхранение на данни

### 🏗️ Упражнения 6-7: Класове и Обекти
- **06_class_basic** - Основен клас с getter/setter методи
- **07_form_with_class** - Форма с класа, 4+ полета и dropdown меню

### 🖼️ Упражнение 8: Галерия
- **08_gallery** - Качване и преглед на снимки, опции за изтегляне и отваряне

### 💾 Упражнения 9-10: База Данни
- **09_form_database** - Запис на форма в MySQL чрез PDO
- **10_database_table_count** - Показване на данни с брой срещанията

### 🛒 Упражнения 11-12: Пълни Системи
- **11_product_catalog** - E-commerce каталог на продукти
- **12_admin_panel** - Администраторски панел с управление на данни

### 🎨 Упражнение 13: Дизайн и Responsive
- **13_responsive_grid** - Bootstrap Grid система с адаптивен дизайн

### ✨ Упражнение 14: Bootstrap Страница
- **14_bootstrap** - Демонстрационна страница с Navbar, Carousel и Cards (Bootstrap)

### 🔍 Упражнение 15: Форма с Regex
- **15_regex_form** - Добре изглеждаща HTML форма с поне две различни полета и сървърна валидация с регулярни изрази

## 💾 База Данни

За упражнения 9, 10, 11 и 12 се нуждаете от MySQL база данни.

### Инсталация
1. Отворете phpMyAdmin: `http://localhost/phpmyadmin`
2. Импортирайте файла `../catalog/seed.sql`
3. Конфигурирайте свързанията в `*/db.php` файловете

### Конфигурация на база
```php
$dbConfig = [
    'host' => '127.0.0.1',
    'dbname' => 'test',
    'user' => 'root',
    'pass' => '',  // обикновено празно за локални среди
    'charset' => 'utf8mb4',
];
```

## 📝 Упражнения по детайли

### 1. Числа - Четни и Нечетни
**Цели**: For цикъл, условни оператори, HTML форми
- Отпечатва числа от 1 до 100 (или  число)
- Четните числа са в червено, нечетните в черно
- Форма за избор на брой числа

### 2. Квадрат с For цикли
**Цели**: Вложени цикли, матрици
- Две вложени for цикъла за рисуване
- Звезди за създаване на визуален квадрат
- Интерактивна форма за размер

### 3. Случайни Числа в Масив
**Цели**: Масиви, функции за работа с масиви
- Генериране на масив със случайни числа
- Показване с цветна разлика
- Динамичен брой елементи

### 4. Div Тагове със Цветове
**Цели**: HTML div, CSS, адаптивност
- Масив с имена на цветове
- Div елементи с цветен фон
- Адаптивен цвят на текста (тъмен/светъл)

### 5. Масив със Сесия
**Цели**: PHP $_SESSION, управление на данни
- Добавяне на елементи в сесийния масив
- Изтриване на отделни или всички елементи
- Персистентност на данни между страниците

### 6. Клас Fruit
**Цели**: ООП, класове, методи
- Дефиниране на клас със 3 атрибута (име, цвят, вкус)
- Getter и setter методи
- Инстанциране и показване на данни

### 7. Регистрационна Форма с Клас
**Цели**: Форми, валидация, дизайн
- 4 текстови полета + 1 dropdown (пол)
- Класа за обработка на данни
- Модерен CSS дизайн
- Показване на резултатите

### 8. Галерия със Снимки
**Цели**: Работа с файлови системи, качване
- Качване на JPG/PNG снимки
- Съхранение в папка `images/`
- Преглед на галерия със снимки
- Опции за изтегляне и отваряне

### 9. Запис на Форма в База Данни
**Цели**: PDO, база данни, безопасност
- HTML форма с полета: име, имейл, възраст, коментар
- Запис чрез PDO на MySQL
- Валидация на вход
- CSRF защита

### 10. Брой Срещания на Записи
**Цели**: SQL запросу, анализ на данни
- Зареждане от произволна таблица
- Намиране на дублирани редове
- Показване на брой на срещанията
- Табличен преглед

### 11. Каталог на Продукти
**Цели**: Пълна e-commerce система
- Управление на продукти (добавяне, редакция, изтриване)
- Категории
- Кошница за пазаруване
- Регистрация и логин

### 12. Администраторски Панел
**Цели**: Управление на данни, CRUD операции
- Логин система със сесии
- Списък на продукти в таблица
- Добавяне и изтриване на продукти
- Управление на категории и поръчки

### 13. Дизайн и Responsive
**Цели**: Bootstrap Grid система и адаптивно оформление
- Три колони в първия ред
- Две колони във втория ред
- Автоматично преустройване на малки екрани

### 14. Bootstrap Страница
**Цели**: Пример за страница със стандартни Bootstrap компоненти
- Navbar
- Carousel (слайдшоу)
- Cards
- Основен дизайн с помощта на CDN

### 15. Форма с Regex
**Цели**: Форма и проверка на данни с регулярни изрази
- Полета за име и имейл
- Серверна валидация чрез preg_match
- Показване на грешки под полетата

Всяко упражнение е независимо и може да се тества поотделно. За упражненията със база данни:

1. Убедете се, че MySQL/XAMPP работи
2. Базата `test` съществува (създайте я ако не)
3. Таблиците са импортирани от `seed.sql`
