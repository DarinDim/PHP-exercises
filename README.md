# 📚 PHP Exercises

Това е колекция от **PHP упражнения** и **пълен е-commerce каталог**, организирани по сложност и раздели.

## 📋 Съдържание

### 🎓 Упражнения (Exercises)
15 различни PHP задачи, покриващи основни и напреднали концепции:

1. **Числа - Четни и Нечетни** - Цикли и условия
2. **Квадрат със For цикли** - Вложени цикли
3. **Случайни числа** - Масиви
4. **Div със цветове** - HTML/CSS интеграция
5. **Масив със Сесия** - PHP $_SESSION
6. **Клас Fruit** - Основни класове
7. **Регистрационна форма** - Формуляри и дизайн
8. **Галерия със снимки** - Качване и управление
9. **Запис в База** - PDO и база данни
10. **Брой срещания** - SQL анализ
11. **Каталог на продукти** - E-commerce система
12. **Администраторски панел** - CRUD операции
13. **Responsive Grid** - Bootstrap Grid + Адаптивен дизайн
14. **Bootstrap Страница** - Демонстрация на Navbar, Carousel и Cards
15. **Форма с Regex** - Добре изглеждаща форма с две полета и валидация с регулярни изрази

### 🛒 Каталог (Catalog)
Пълна e-commerce система с:
- Управление на продукти
- Работа с категории
- Кошница за пазаруване
- Регистрация и логин
- Администраторски панел

## 🚀 Начало

### Изисквания
- PHP 7.4+
- MySQL 5.7+
- XAMPP или другия локален сървър

### Инсталация

1. **Клониране на репото**
```bash
git clone https://github.com/yourusername/PHP-exercises.git
cd PHP-exercises
```

2. **Копиране на файловете в XAMPP**
```bash
cp -r exercises /path/to/xampp/htdocs/
cp -r catalog /path/to/xampp/htdocs/
```

3. **Конфигурация на база данни**
- Отворете phpMyAdmin: `http://localhost/phpmyadmin`
- Импортирайте `catalog/seed.sql`
- Конфигурирайте `catalog/db.php`

4. **Стартиране**
- Упражнения: `http://localhost/exercises/`
- Каталог: `http://localhost/catalog/`
- Админ панел: `http://localhost/catalog/admin/`
